# SilverOS/main
An OS simulator as a phone.

## Installing
To install open a terminal and run the following command.

```bash
curl -O https://raw.githubusercontent.com/HttpAnimation/SilverOS/main/install.sh
chmod +x install.sh
./install.sh
```
